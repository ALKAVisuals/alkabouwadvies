# Complete Site Analyse - v0.01

## Analyse Checklist (per pagina)

### 1. Technische Kwaliteit
- [ ] HTML validatie (geen syntax errors, correcte nesting)
- [ ] Meta tags compleet (title, description, viewport)
- [ ] Canonical URL aanwezig
- [ ] Structured data / JSON-LD schema
- [ ] Favicon en app icons
- [ ] Preload directives voor critical resources

### 2. SEO & Content
- [ ] Title tag optimal (50-60 chars)
- [ ] Meta description aanwezig (120-160 chars)
- [ ] H1 tag uniek en relevant
- [ ] Content relevantie en diepte
- [ ] Interne links naar andere relevante pagina's
- [ ] Alt-teksten bij afbeeldingen
- [ ] URL structuur (slug optimal)
- [ ] XML sitemap status

### 3. Performance
- [ ] Image optimalisatie (format, grootte, lazy loading)
- [ ] Critical CSS inline of preloaded
- [ ] JavaScript bundling en minificatie
- [ ] Font loading optimalisatie
- [ ] Unused CSS/JS detection

### 4. Toegankelijkheid (a11y)
- [ ] Skip links / focus management
- [ ] ARIA labels en roles
- [ ] Color contrast ratios
- [ ] Form labels en error handling
- [ ] Landmark roles (main, nav, footer, etc.)

### 5. Design & UX Consistency
- [ ] Navigatie consistent met homepage
- [ ] Footer consistent met homepage
- [ ] Typography scale consistent
- [ ] Button styling consistent
- [ ] Mobile responsive design

### 6. Security
- [ ] HTTPS enforcement
- [ ] CSP headers (indien van toepassing)
- [ ] Form security (honeypot, validatie)
