# ALKA Bouwadvies Website

**Live URL:** https://alkavisuals.github.io/alkabouwadvies/  
**Repository:** github.com/ALKAVisuals/alkabouwadvies  
**Versie:** 0.2 | **Laatste update:** 22-05-2026

## Overzicht

33 statische HTML pagina's (~93.600 regels) voor ALKA Bouwadvies — bouwbegeleiding, vergunningen en advies in Nederland.

## Structuur

- **1 homepage** (index.html)
- **15 dienstpagina's** (dakkapel, aanbouw, erker, bijgebouw, etc.)
- **11 stadspagina's** (Apeldoorn, Arnhem, Breda, Den Haag, Eindhoven, Groningen, Nijmegen, Rotterdam, Tilburg, Utrecht, Amsterdam)
- **3 juridische pagina's** (privacybeleid, algemene voorwaarden, cookiebeleid)
- **1 blog** (blog.html + 1 artikel)
- **Overige:** over-ons, contact, testimonials, aankoopadvies

## SEO & Technisch

- ✅ Sitemap.xml + robots.txt
- ✅ Schema.org (LocalBusiness, FAQPage, BlogPosting)
- ✅ Meta titles, descriptions, keywords op alle pagina's
- ✅ Open Graph + Twitter Cards
- ✅ Canonical URLs
- ✅ Responsive (breakpoints: 1024px, 768px)
- ✅ Mobile hamburger menu + sticky CTA
- ✅ Skip links, ARIA labels, alt-teksten
