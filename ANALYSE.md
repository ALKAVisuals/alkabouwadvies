## 2. TECHNISCHE ANALYSE

### 2.1 Structuur
- **Type:** Statisch HTML (geen CMS)
- **Talen:** Nederlands (nl)
- **Responsive:** Ja (breakpoints: 1024px, 768px)
- **Mobile navigatie:** Hamburger menu
- **Mobile sticky CTA:** Ja (verschijnt bij scroll >600px)

### 2.2 Performance
- **Laadtijd:** Goed (statisch HTML, geen server-side processing)
- **CSS:** Inline in <style> tags (geen externe CSS files)
- **JS:** Inline <script> tags
- **Afbeeldingen:** PNG formaat in /images/ map
- **Fonts:** Google Fonts (Inter)
- **CDN:**cdnjs.cloudflare.com (GSAP, Font Awesome, Lenis)

### 2.3 SEO Basis
- **Meta titles:** Aanwezig op alle pagina's
- **Meta descriptions:** Aanwezig op alle pagina's
- **Meta keywords:** Aanwezig op alle pagina's
- **Open Graph:** Aanwezig op alle pagina's
- **Twitter Cards:** Aanwezig op alle pagina's
- **Canonical URLs:** Aanwezig op alle pagina's
- **Hreflang:** Niet aanwezig
- **Sitemap.xml:** Niet aanwezig
- **Robots.txt:** Niet aanwezig
- **Schema.org markup:** Niet aanwezig

### 2.4 Toegankelheid
- **Skip links:** Aanwezig
- **ARIA labels:** Aanwezig op navigatie
- **Alt teksten:** Aanwezig op afbeeldingen
- **Kleurencontrast:** Goed (donkere tekst op lichte achtergrond)
- **Lettergrootte:** Goed (rem-based)
